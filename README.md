# e-Sahayta – SOS Emergency Assistance App

A real-time emergency response mobile application built using Flutter and Firebase.

## Features
- SOS emergency alert system
- Live location tracking (Google Maps API)
- Geo-fencing for safety zones
- Sensor-based triggers (shake detection, voice activation)
- Real-time notifications using Firebase Cloud Messaging (FCM)

## Tech Stack
- Flutter (Dart)
- Firebase (Authentication, Firestore, Realtime DB)
- Google Maps API

## How It Works
- User triggers SOS manually or via sensor
- Location is captured and sent in real-time
- Emergency contacts are notified instantly

## Future Improvements
- AI-based threat detection
- Offline emergency mode
